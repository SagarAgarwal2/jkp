Project Overview

Digital Twin Market Engine is an advanced simulation platform built for FMCG (Fast-Moving Consumer Goods) companies to model and predict product launch outcomes. Running entirely in the browser using TensorFlow.js, the system simulates 25,000 synthetic customer personas and uses machine learning to predict purchase probability, customer segmentation, churn risk, and optimal pricing strategies.

The platform demonstrates the power of client-side machine learning, eliminating the need for backend infrastructure while delivering sophisticated predictive analytics. Built with React for a responsive interface and Chart.js for data visualization, the simulator provides immediate insights into market dynamics.

Key ML capabilities include:

  • Neural Networks: Deep learning models for purchase probability prediction
  • K-Means Clustering: Customer segmentation based on behavior patterns
  • Regression Models: Price elasticity and demand forecasting
  • Churn Prediction: Identify customers at risk of switching products

Key Features

  • Browser-based ML: All machine learning runs client-side with TensorFlow.js
  • Synthetic Persona Generation: Creates 25,000 realistic customer profiles with diverse characteristics
  • Purchase Probability Prediction: Neural network predicts likelihood of purchase for each persona
  • Customer Segmentation: K-Means clustering groups customers into meaningful segments
  • Churn Risk Analysis: Identifies customers likely to stop purchasing
  • Pricing Optimization: Regression models find optimal price points
  • Interactive Dashboards: Real-time visualization of simulation results using Chart.js
  • Scenario Testing: Compare different product launch strategies
  • Export Capabilities: Download simulation data and insights

Technical Implementation

The platform leverages cutting-edge web technologies for ML inference:

  • TensorFlow.js: Complete ML framework running in the browser
  • Neural Network Architecture: Multi-layer perceptrons with dropout and batch normalization
  • K-Means Implementation: Custom clustering algorithm optimized for browser performance
  • Web Workers: Parallel processing for faster simulations
  • IndexedDB: Local storage of model weights and simulation results
  • Chart.js: Dynamic, responsive visualizations for insights
  • React Context API: Efficient state management for ML models

The system generates synthetic data using probabilistic models that mirror real-world customer behavior, including demographics, purchase history, price sensitivity, and brand loyalty factors.

Use Cases

  • Product Launch Planning: Simulate market response before actual launch
  • Pricing Strategy: Find optimal price points for maximum revenue
  • Market Segmentation: Identify and target high-value customer segments
  • Churn Prevention: Proactively address customer retention risks
  • A/B Testing: Compare different marketing and pricing strategies
  • Demand Forecasting: Predict sales volumes under various scenarios
