Project Overview

As Founder & Developer of SaviHer, I built a comprehensive women's safety application that prioritizes speed and reliability in emergency situations. The app achieves a remarkable 2.5-second response time from SOS trigger to emergency contact notification, making it 70% faster than traditional emergency alert systems.

SaviHer has been incubated at Bhamasha Techno Hub and is currently in the process of applying for iSTART state funding, recognizing its potential impact on women's safety across India.

The app leverages multiple technologies to ensure maximum reliability:

  • Flutter/Dart: Cross-platform mobile development for iOS and Android
  • Firebase: Real-time database and cloud storage for instant data sync
  • GCP: Image processing and machine learning capabilities
  • Twilio: SMS and voice call integration for emergency notifications
  • Location Services: Real-time GPS tracking with offline capabilities

Key Features

  • Instant SOS Alert: One-tap emergency button with 2.5-second response time
  • Live Image Capture: Automatically captures and uploads photos to cloud storage
  • Real-time Location Tracking: Continuous GPS tracking shared with emergency contacts
  • Multi-channel Notifications: SMS, push notifications, and automated voice calls
  • Offline Mode: Core features work without internet connectivity
  • Trusted Circle: Manage up to 10 emergency contacts with priority levels
  • Fake Call Feature: Simulate incoming calls to exit uncomfortable situations
  • Safe Zone Alerts: Geofencing notifications when entering/leaving designated areas
  • Audio Recording: Background audio recording during emergencies
